DOM

Document Object Model

Dom is an interface where you can manuplate tags attributes and style through the javascript.Once the Docume is created dom tree will be created parallely.

​

In the Dom tree every end point is a node. we have a diffent type of nodes

​

Types of nodes

​
  1. Elemental node : When the node is a tag then we consider it as element
  2. Text node : when the node is a normal text then we call it as text node
  3. Attribute node : when the node is a attribute then we call it as attribute node
